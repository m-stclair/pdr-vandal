Basic Usage

To open an image, click the "Open With PDR" button and select the file or files you wish to open. If this is the first image you've opened with PDR in the session, PDR will install (10-20 seconds). After opening, the image will display in the main pane. You can zoom the image in and out using mouse or touchpad scroll. When zoomed, you can pan the image by dragging.

Switching Bands and Objects

If the product you've uploaded has more than one image (like a FITS file with multiple image HDUs), you can switch between them using the "object" dropdown in the upper right-hand corner. If an image has multiple bands/channels, you can swap between them using the "bands" dropdown immediately below the "object" dropdown. If an image has 3 or 4 bands, the "bands" dropdown will also have an "RGB" option that displays the first three bands stacked to RGB.

Effects

The right-hand pane also contains an "effect picker" and an "effect stack" for visual effects like colormapping. You can add effects by clicking on them in the picker. You can open an effect for editing by clicking on it, and remove it by clicking its X button. The "Input Stretch" effect is always pinned at the top of the stack. This effect can clip or stretch the input range for display before rendering or applying other effects. Images with wide value ranges may not be visible without manipulating this effect.

Image Export

You can save the displayed image, including pan/zoom and visual effects, by clicking the "Export PNG" button in the top bar.

Browser Image Loading

If you want to open a "desktop" format image like a JPEG or PNG, you can open it with standard browser features instead of PDR by clicking the "Open With Browser" button in the top bar. Desktop images will load more quickly this way, but will not support channel flipping or some PDR features like accessing secondary images in MPOs.

Privacy

pdr-vandal runs locally in your browser. It does not upload files to a server for processing. Your files never leave your computer.
